## 🎮 Como Usar

### Na Tela Principal
1. **Clique em "Iniciar"** (botão verde grande)
2. **Ajuste o BPM** com o slider
3. **Escolha subdivisão** (semimote, corda, etc)
4. **Ajuste o afinação** (Hz)
5. **Pratique** com feedback visual e haptic

### No Setlist
1. **Clique em "Setlist"** no menu
2. **Adicione** uma música com nome, BPM, subdivisão
3. **Salve**
4. **Selecione** a música e **Clique para carregar**
5. **Pratique** rapidamente

## 🐛 Solução de Problemas

### Não Está Funcionando?

**1. Verifique se está rodando:**
- O terminal deve mostrar "ready in XXX ms" e "Local: http://localhost:5000"

**2. Se a porta já está em uso:**
- Outro processo está usando a porta 5000
- Fecha os outros serviços usando essa porta
- Ou use outra porta: `npm run dev -p 8000`

**3. Se não aparece nada:**
- Verifique se o Node.js está instalado
- Verifique se as dependências foram instaladas: `npm install`

**4. Erro de TypeScript:**
- Verifique o terminal para detalhes do erro
- Certifique-se de usar TypeScript 5.4+
