## 🎯 How to Use

### Practice (3 Steps)
1. **Create Song:**
   - Click "Setlist"
   - Add name, BPM, subdivision
   - Save

2. **Practice:**
   - Select song
   - Click "Start"
   - Practice with visual & haptic feedback

3. **Done:**
   - Stop metronomo
   - Review progress

### Organize (3 Steps)
1. **Create Setlist:**
   - Add multiple songs
   - Each with own settings

2. **Organize:**
   - Order songs for practice
   - Add notes for special instructions

3. **Share:**
   - Export as JSON
   - Share with bandmates
